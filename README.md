# One Click Image Download

One Click Image Download is a Firefox and Zen extension that downloads the largest loaded image on the current page with one toolbar click.

## How to use

1. Open a page that contains images.
2. Click the extension button in the browser toolbar.
3. The extension finds the largest loaded image on the page and downloads it automatically.

The extension also tries to preserve the page referrer when downloading images from sites that block direct image requests.

## Notes

- It works with normal page images, including responsive images selected by the browser.
- It does not scan CSS background images, canvas drawings, or video posters.
- It cannot run on Firefox system pages or other restricted pages.
- If a file with the same name already exists, Firefox saves a uniquely named copy.
